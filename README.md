# Velocity-Racing-grupo-14

#En el siguiente proyecto se busca desarrollar una herramienta que permita mejorar la adminstración de la información deportiva de Velocity Racing Team.
#El sistema permite registrar y administrar datos de los pilotos. Siendo estos mismos los siguientes:
# 1. Nombre del piloto
# 2. Número de monoplaza
# 3. Escudería asociada
# 4. Puntos acumulados en el campeonato
# 5. Tiempo promedio por vuelta
# 6. Presupuesto asignado al piloto
# 7. Cantidad de abandonos en la temporada

#También cabe destacar que cuenta con las siguientes funcionalidadees
# 1. Registro de pilotos
# 2. Eliminación de pilotos
# 3. Modificación de tiempo o puntos de pilotos
# 4. Producción de un informe general
